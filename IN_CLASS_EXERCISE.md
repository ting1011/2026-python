## 3. 整理方式

### 資料來源

1. **`ENROLLMENT.md`**：全班 55 位學生學號名單。
2. **`gh pr list --limit 100`**：抓取 repo 內所有 open PR。

### 篩選邏輯

**課堂練習 PR 的判斷條件（同時符合）：**
- PR 建立日期為課堂當日
- PR 標題或分支名稱含有學號或明確的課堂標記（`MMDD`、`class` 等）
- **排除**分支為 `submit/week-XX` 且標題符合作業格式的 PR

**例外處理：**
- 建立日期為隔日但標題含課堂日期者，判定為補交，狀態標記 ⚠️
- 學號不在 enrollment 名單者，列入「非 Enrollment 名單」區塊
- 教師 / 助教 / 測試用 PR 直接排除

### 學號對應方式

1. PR 標題中直接含學號（如 `0305-1114405010`）
2. 分支名稱含學號（如 `d14405011-sudo:0305-1114405011-class`）
3. GitHub 帳號名稱含學號（如 `1114405040Shihminhung`）

無法辨識學號者列入「非 Enrollment 名單」區塊另行記錄。
