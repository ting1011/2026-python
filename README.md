# 2026 Python CPE Weekly Practice

以開源專案常見結構整理每週課程與解題進度，方便協作、版本控管與後續擴充。

## Repository Structure

```text
.
├── docs/
│   └── COURSE_PLAN.md
└── weeks/
    ├── week-01/
    ├── week-02/
    ├── ...
    └── week-18/
```

## Usage

- 每週資料放在 `weeks/week-XX/`。
- 題目程式碼放在各週的 `solutions/`。
- 每週說明與作業放在各週 `README.md`。
- 題號參考：<https://yuihuang.com/cpe-level-1/>
