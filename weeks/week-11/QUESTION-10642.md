## 題目敘述

在一張圖上，每個圓圈都有一個**笛卡爾坐標**。你可以按照圖中**箭頭指示的路徑**從一個圓圈移動到另一個圓圈。

計算**步數**的公式為：

$$\text{總步數} = \text{經過的中間圓圈數} + 1$$

例如：從 **(0, 3)** 到 **(3, 0)**，需要經過中間圓圈 (1, 2) 和 (2, 1)，所以總步數 = 2 + 1 = **3**。

給定一個**起始座標**和一個**目標座標**，請計算從起點到達終點**所需的最少步數**。

（注意：不能走回頭路，即不能逆著箭頭方向移動）

## 輸入說明

- 第一行為整數 **n**（測試資料組數，0 < n ≤ 500）。
- 接下來 n 行，每行包含四個整數 **x1, y1, x2, y2**（0 ≤ 各整數 ≤ 100000）：
  - **(x1, y1)**：起始座標
  - **(x2, y2)**：目標座標

## 輸出說明

對每組測試資料，輸出格式如下：

```
Case K: steps
```

其中 K 為組號（從 1 開始），steps 為所需的最少步數。
