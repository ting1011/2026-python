## 題目敘述

**完全平方數（perfect square）** 是指平方根為整數的正整數，例如 1, 4, 9, 16, 25, 81, …

給定兩個整數 **a** 和 **b**，求閉區間 **[a, b]** 中**完全平方數的個數**。

## 輸入說明

- 每行兩個整數 **a** 和 **b**（**1 ≤ a ≤ b ≤ 100,000**）。
- 以 **a = 0, b = 0** 結束輸入（不需處理）。
- 最多 201 行輸入。

## 輸出說明

每行輸出一個整數，為 [a, b] 中**完全平方數的個數**。
