## 題目敘述

給定正整數 **N**，計算所有滿足 **1 ≤ i < j ≤ N** 的整數數對之 **GCD（最大公因數）的總和**：

$$G = \sum_{i=1}^{N-1} \sum_{j=i+1}^{N} \gcd(i,\ j)$$

## 輸入說明

- 每行一個正整數 **N**（**2 ≤ N ≤ 500**）。
- 以 **N = 0** 結束輸入（不需處理）。

## 輸出說明

每個 N 輸出一行，為 **G 的值**。
