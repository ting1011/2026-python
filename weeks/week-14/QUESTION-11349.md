## 題目敘述

給定一個 **n×n 的方陣 M**，判斷它是否為「**對稱矩陣**」。

本題的「對稱」定義為：
1. 矩陣所有元素均為**非負數**（≥ 0）
2. 矩陣關於**中心點對稱**，即對所有 i, j 滿足：
   $$M[i][j] = M[n+1-i][n+1-j]$$

（注意：這裡是中心對稱，**不是**一般的轉置對稱 M[i][j] = M[j][i]）

## 輸入說明

- 第一行為測試資料組數 **T**（T ≤ 300）。
- 每組測試資料：
  - 第一行格式為 `N = n`（矩陣維度，0 < n ≤ 100）
  - 接下來 n 行，每行 n 個整數（元素範圍：−2³² ≤ Mᵢⱼ ≤ 2³²）

## 輸出說明

每組測試資料輸出一行：

- 若為對稱矩陣：`Test #t: Symmetric.`
- 若不是：`Test #t: Non-symmetric.`

（t 為測試組號，從 1 開始）
