
# 8 容器操作與推導式

## 推導式（Comprehensions）

### 列表推導式
```python
[x for x in data if x > 0]  # 建立一個新列表，僅保留 data 中大於 0 的元素
```

### 字典推導式
```python
{k: v for k, v in d.items()}  # 將 d 的所有鍵值對複製到新字典
```

### 生成器表達式
```python
(x * x for x in nums)  # 產生一個生成器，依序產生 nums 中每個元素的平方
```

---

## 用途（對應第一章範例）：

- 過濾序列（1.16）：用列表推導式快速過濾資料
- 字典子集（1.17）：用字典推導式過濾部分 key
- `sum(...)` / `min(...)` / `join(...)`：生成器表達式可直接作為聚合函式的輸入，節省記憶體
