# 手寫程式紀錄

## 10189 Minesweeper
1. 先讀入 n, m，直到 0 0 結束
2. 用二維陣列存地圖
3. 逐格檢查，遇到 '*' 就把周圍 8 格 +1
4. 輸出格式注意 Field # 與空行

## 10190 自動傘雨量
1. 讀入 N, W, T, V
2. 依序讀入每把傘的 x, l, v，記錄覆蓋區間
3. 合併所有傘的覆蓋區間
4. 算出未覆蓋長度，乘 T*V 得答案

## 10193 arctan 分解
1. 讀入 a
2. 枚舉 b 從 a+1 開始，c = a*b/(b-a)
3. 若 c 為整數且 c>=b，取最小 b+c

## 10221 Satellites
1. 讀入 s, a, 單位
2. 單位為 min 則 a/60，轉成度
3. 轉成弧度，若大於 pi 則用 2pi-a
4. 弧長 r*a，弦長 2*r*sin(a/2)

## 10222 Decode the Mad man
1. 建立鍵盤字串
2. 逐字元左移 3 位，空白不變
3. 輸出解碼後字串
