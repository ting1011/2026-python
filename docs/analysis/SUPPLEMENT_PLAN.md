## 🔗 備選補充來源

### 1. UVA Online Judge (最可靠)
URL: https://uva.onlinejudge.org/external/{UVA_ID}.pdf

**優點**:
- 官方來源，最權威
- PDF 格式穩定可靠
- 題目敘述完整

**劣點**:
- 需要手工複製文字
- 部分題目可能需要登入

### 2. Codeforces
URL: https://codeforces.com/problemsets/acmsguru/problem/{ID}

**優點**:
- 備選平台
- 通常有完整敘述

**劣點**:
- 非所有題目都收錄
- 格式可能不同

### 3. Yui Huang 題解
URL: https://yuihuang.com/zj-{ZJ_ID}/

**優點**:
- 中文解釋
- 已有部分題解

**劣點**:
- 不是所有題目都有
- 主要是解題思路

### 4. LeetCode/LintCode
URL: 搜尋題目名稱

## 🔧 補充方式

### 方式 1: 直接從 UVA PDF 複製
1. 打開 https://uva.onlinejudge.org/external/{UVA_ID}.pdf
2. 複製題目敘述 → 貼到 `## 題目敘述`
3. 複製輸入說明 → 貼到 `## 輸入說明`
4. 複製輸出說明 → 貼到 `## 輸出說明`

### 方式 2: 使用 Python 腳本補充
```python
# 編輯 fill_missing_content.py
# 新增 UVA_TO_MANUAL 映射（手工輸入內容）
# 執行補充
```

### 方式 3: 批量編輯
1. 編輯每個缺失的 QUESTION-*.md
2. 移除 "說明請見上方連結"
3. 填入完整內容
